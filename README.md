# Badrudin AI OS

Единая цифровая операционная система для управления ООО «Экстра-Элит»:
строительными объектами, проектно-изыскательскими работами, архитектурой и
дизайном, финансами, снабжением, документооборотом, маркетингом и контролем
исполнения.

Система собирает разрешённые данные из CRM, мобильных форм, почты и официальных
каналов, анализирует их с помощью специализированных ИИ-агентов, готовит проекты
задач и решений и контролирует исполнение — при этом критические действия всегда
остаются под контролем человека.

> Главный принцип: система ускоряет работу, повышает прозрачность и дисциплину
> исполнения, сохраняя окончательный контроль за человеком.

## Назначение

- единый центр управления организацией, проектами и объектами;
- работа нескольких специализированных ИИ-агентов под контролем человека;
- обязательное согласование критически важных действий (уровень риска R4);
- сбор данных из почты, форм, файлов, официальных мессенджеров и CRM;
- контроль сроков, обратной связи и подтверждений выполнения;
- ведение отдельного цифрового контура по каждому объекту;
- хранение полной истории решений и действий (журнал аудита).

## Ключевые принципы

- **Человек в контуре.** Критические действия уровня R4 (платежи, договоры,
  кадровые решения, публикации, удаление и массовый экспорт) выполняются только
  после подтверждения уполномоченного человека.
- **Documentation First.** Бизнес-смысл документов не меняется без записи в
  `DOCS/DECISIONS.md`.
- **Технологический стек MVP:** FastAPI, PostgreSQL, Next.js, n8n
  (см. `DOCS/DECISIONS.md`, решение D-004).
- **Единый источник MVP:** `DOCS/CLAUDE.md`, синхронизированный с `DATABASE.md`,
  `ARCHITECTURE.md` и `ROADMAP.md` (решение D-003).
