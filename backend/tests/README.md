# Тесты backend

Каркас автотестов Этапа 1 (T-1.I1).

## Запуск

```bash
cd backend
pip install -e ".[dev]"
pytest
```

## Принципы

- Тесты изолированы: используется отдельная тестовая БД SQLite в памяти
  (фикстуры `db_engine` / `db_session` / `client` в `conftest.py`); рабочая база
  не затрагивается.
- Тестовые данные обезличены, реальные ПДн и секреты не используются (D-011).
- Маркер `integration` помечает тесты, требующие внешних сервисов (PostgreSQL,
  Redis, MinIO); в окружении без сервисов такие тесты не запускаются.
- Проверки миграций выполняются через Alembic на SQLite; на PostgreSQL —
  в CI/окружении с БД (T-1.I2).
